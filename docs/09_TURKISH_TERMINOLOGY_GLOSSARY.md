# Farm ERP Turkish Terminology Glossary

## Purpose

This document is the official terminology reference for the Farm ERP project.

Its purpose is to ensure consistent wording across the user interface, documentation, translations, and future development.

This glossary is the Single Source of Truth (SSOT) for all English → Turkish translations used in the project.

---

## Terminology Rules

### Rule 1 — Avoid ambiguous terms

Do not use generic words without context.

Avoid:

- Total
- Current
- Recent
- Latest

Prefer explicit wording instead.

Examples:

- Today's Milk
- Last 7 Days Milk
- Last 30 Days Milk
- Lifetime Milk Production
- Recent Records
