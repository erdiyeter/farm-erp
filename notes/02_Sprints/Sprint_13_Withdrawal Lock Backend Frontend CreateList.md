# Sprint 13 — Withdrawal Lock Frontend

## Sprint Goal

Implement the first frontend package for the Withdrawal Lock module.

This sprint connects the completed Withdrawal Lock backend to the React frontend and provides basic record creation and listing functionality.

---

## Scope

### Implemented

- Withdrawal Lock API client
- Withdrawal Lock list page
- Withdrawal Lock create form
- Navigation integration
- Route integration
- Manual frontend testing

### Out of Scope

- Detail page
- Edit page
- Delete action
- Filtering
- Search
- Alarm integration
- Dashboard integration

These items are planned for Sprint 14.
